# Mark My Words — Writers' Room

This directory contains all active and completed writing projects managed by the Mark My Words multi-agent system.

## Structure

- `brand/` — Brand guidelines used by all agents during piece workflows
- `pieces/` — Active piece folders (one per codename)
- `published/` — Final drafts and image prompts ready for publishing
- `research/` — Shared research notes across all pieces
- `index/` — Post archive index
- `cadence/` — Publishing calendar

## Usage

See `./CLAUDE.md` at the project root for trigger syntax and workflow overview.
