# Readability Standard (Flesch-Kincaid Mental Model)

This document provides a shared mental model for auditing and improving content. All skills must use this standard for calibration.

---

## Core Rules

1. **Word Count**: Average fewer than 20 words per sentence.
2. **Architecture**: Use Subject-Verb-Object (SVO) order.
3. **Syllable Control**: Use short, common words instead of jargon.
4. **Data Over Diction**: Describe observations using direct metrics.
5. **Clarity Score**: Target a Flesch-Kincaid score of 60-70.

---

## Readability Self-Audit Formula (Mental Model)

1. **Length**: Is the average sentence over 20 words?
2. **Syllables**: Do paragraphs contain more than three multi-syllabic words?
3. **Voice**: Does the subject perform the action?
4. **Scoring**:
   - **90-100**: Too simple.
   - **60-70**: **Optimal Target.**
   - **0-50**: Too complex.

---

## Calibration Examples

### ❌ Complex
> "In order to facilitate a more robust architectural orientation, we are leveraging serverless paradigms to significantly mitigate the operational overhead typically associated with infrastructure management."

### ✅ Clear
> "We use serverless code to cut costs. This lets us build features instead of managing servers."
